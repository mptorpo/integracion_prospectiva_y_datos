# integracion_prospectiva_y_datos
En este repositorio se verá cómo fue empleado la analítica de datos y las diferentes tecnologías de la información. Además, seleccionaremos los métodos de analítica de datos más adecuados de acuerdo con la información disponible y el tipo de decisión a tomar en las organizaciones. También, identificaremos los principios de la lógica borrosa en el modelamiento de las variables aleatorias como variables lingüísticas; integrando información cualitativa proveniente de expertos como soporte a la toma de decisiones de las organizaciones en contextos bajo incertidumbre. Comprendiendo los conceptos de la teoría de las decisiones en la creación de escenario prospectivos que permitan comprender a futuro el entorno.
